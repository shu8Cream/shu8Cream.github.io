未完成のまま長らく放置していた自分のサイトを Gemini[^1] を使って Astro で作り直しました。
フレームワークの選択から相談して、「Astro」「Next.js」「Hugo」の3候補から一番おすすめされた「Astro」を選択しました。

公開日時点では、制作物の一覧機能とブログ機能の2つを実装しました。
最低限ほしい機能というか思いついたものしか実装できていないため、今後もアップデート予定です。

[^1]: 使用モデルは Gemini3 Pro preview 版です

## トップページについて
トップでひと際目を引くアートワークは[Okazz](https://x.com/okazz_)さんの作品です。好きな作品だったので使用させていただいております。
作品は[こちらのサイト](https://openprocessing.org/sketch/2326097)で公開されており、他の作品も見ることができます。
p5.js[^2]で書かれています。

[^2]: 初心者でも直感的に、ブラウザ上で図形描画やアニメーションなどのコードが書けるJavaScriptライブラリ。ProcessingのJavaScript版。この動画がわかりやすいです。[デザイナーが語る「p5.js」の魅力｜好き語り](https://www.youtube.com/watch?v=lQyZvbmxV_M)


## 機能について
どうせしばらくしたら使い方を忘れるので、追加した機能をここにまとめておけば後で見返せるという魂胆です。

### 数式を描画できる
競プロやLLMで調べたことを書いておこうと思っているので、数式が書けると嬉しいので機能追加しました。

インライン数式は $E = mc^2$ のように書きます。
ブロック数式は `$$` で囲みます：

$$
y = \sum_{i}^{N}{x_i}
$$

$$
\frac{1}{2\pi i} \oint_\gamma f(z) \, dz = \sum_{k=1}^n \text{Res}(f, a_k)
$$

行列も書けます：

$$
A = \begin{pmatrix}
a & b \\
c & d
\end{pmatrix}
$$

とっても嬉しい！

### コードブロック
適当に導入したマークダウンの描画ライブラリだと、コードの縦幅ピッタリ分しか黒背景がなくて、コピーボタンもありませんでした。
この苦情をGeminiへそのまま伝えて、新しいプラグイン「Expressive Code」を導入することで解決しました。
見た目はもちろんのこと、ファイル名を付けることやハイライト機能も付きました。

```javascript
console.log("Hello, Astro!");
```

<br/>

```python title="app.py" {3-4} del={5} ins={6}
def hello():
    print("ここはハイライトされません")
    print("ここがハイライトされます！")
    print("ここも！")
    print("Apple!")
    print("Banana!")
```

<br/>

```bash
echo "This terminal frame has no title"
```

<br/>

```powershell title="PowerShell terminal example"
Write-Output "This one has a title!"
```

こんなにリッチで使いこなせるのかは不明です
